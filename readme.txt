================
= Instructions =
================

To run this:
 * You should have installed Node
 * Created an app on https://discordapp.com/developers/applications/
 * Note down the bot token

 * Create a auth.json file with the contents:
   {
    token: "<BOTTOKENKEEPSECRETNEVERPUSH>"
    prefix: "ss~"
   }

 * Add the client ID from the application to the link below, remove the <>
   https://discordapp.com/oauth2/authorize?&client_id=<CLIENT ID>=bot&permissions=1

 * Run the "npm install" to install the packages found in package.json

 * Go to dir of your discordbot and run "node bot.js"

 * Go to the discord app and type ss~commands
